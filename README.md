## Features
- **Timer Mode** ‚Äì preset and custom countdown timers (5, 10, 15, 25, 50 minutes)  
- **Stopwatch Mode** ‚Äì track elapsed time with lap recording  
- **Dark/Light Theme** ‚Äì toggle between cyberpunk dark and clean light themes  
- **Responsive Design** ‚Äì works on desktop & mobile  
- **Visual Feedback** ‚Äì dynamic animations reacting to state  
- **Sound Effects** ‚Äì timer ticks, completion alerts, lap recordings (mute/unmute toggle)  

---

## Usage

### Timer
1. Select the "Timer" tab  
2. Choose a preset time or set a custom time  
3. Start, pause, or reset the timer  
4. Completion triggers an alert sound  

### Stopwatch
1. Select the "Stopwatch" tab  
2. Start, pause, or reset the stopwatch  
3. Click flag button to record laps  
4. View laps in the scrollable list
